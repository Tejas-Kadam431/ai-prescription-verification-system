# AI-Powered Prescription & Medicine Verification System

This project aims to build a real-world healthcare system that:
- Extracts text from medical prescriptions using OCR
- Parses medicine details using NLP
- Checks drug interactions and safety rules
- Provides a pharmacist/admin verification workflow

# Tech Stack 
- Frontend: React
- Backend: Node.js, Express.js
- AI Service: Python (OCR+NLP)
- Database: MongoDB
- Cloud & DevOps: Docker, AWS (planned)
